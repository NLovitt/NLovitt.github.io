JS Snippets

Hosted JavaScript snippets that can be injected into any page via URL.

Available scripts

ScriptDescriptionInject URL
example.js Shows an alert with the current page title. loader.html?src=js/example.js

How to inject a script

  1. Bookmarklet — drag this link to your bookmarks bar: Run example.js
  2. Script tag — paste in the browser console:
    var s=document.createElement('script');s.src='https://nlovitt.github.io/js/example.js';document.body.appendChild(s);
  3. Loader page — open loader.html?src=js/example.js to run the script in an isolated page.
